The Federal Bureau of Investigation (FBI), CISA, and NSA are releasing this joint Cybersecurity Advisory to provide details on the fictional APT group 'ShadowVeil,' based on a simulated attack scenario for educational purposes. The actors likely conduct reconnaissance [T1589] to identify high-value targets, such as energy and financial organizations, though specific reconnaissance logs were not captured in this exercise. Initial access was achieved through spearphishing emails [T1566.001] containing malicious attachments disguised as PDF documents, which, when opened, executed a payload that created scheduled tasks [T1053] for persistence and leveraged Process Doppelg√§nging [T1055.013] to hide malicious code in system processes by modifying memory allocation. On macOS systems, if the malware gained root permissions, it established persistence by creating a launch daemon [T1543.004] in the /Library/LaunchDaemons folder. To evade detection, the code executed virtualization/sandbox evasion techniques [T1497], such as querying security logs with commands like 'wevtutil ql Security "*[System[(EventID=4624)]]"' to analyze logon events, and installed a custom UEFI module [T1542.001] to run malicious firmware before OS boot. For lateral movement, the actors used internal spearphishing [T1534] after stealing credentials and collected web session cookies [T1550.004] to bypass multi-factor authentication on internal applications. Data exfiltration involved embedding command and control communications within digital images using steganography [T1001.002] and transferring sensitive documents via physical USB devices [T1052]. Finally, to inhibit recovery, the malware deleted volume shadow copies [T1490] on compromised hosts. System logs from the fictional victim network showed normal traffic patterns interspersed with anomalous outbound connections to IP addresses in Eastern Europe, but these were not directly linked to the techniques above. The organization's background includes a recent merger, which may have introduced security gaps, though this is speculative and not part of the attack chain.
